# Worker Pool 练习题

## 练习1：基础 Worker Pool（中等难度）

**文件**：`exercise1.md`

**核心要求**：
- 实现固定数量的 Worker
- 处理 10 个任务（计算平方）
- 正确关闭 channel，无死锁

**预计完成时间**：15-20 分钟

## 练习2：带超时和错误处理的 Worker Pool（困难）

**文件**：`exercise2.md`

**核心要求**：
- 实现 Worker Pool + 超时控制
- 处理任务失败和超时
- 统计成功/失败/超时数量

**预计完成时间**：30-40 分钟
